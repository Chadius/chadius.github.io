Part 4
31. body div.wrapper > aside ~ aside
An aside element
Where any sibling is an aside element
Whose parent has a class of "wrapper"
That is a DIV element
Whose descendant is a BODY element

32. nav ul li:nth-child(odd) a:hover
A link that is Hovered over
That is part of an A element
That is the odd numbered child
That is an LI element
Whose descendant is a UL element
Whose descendant is a NAV element

33. #content li + li:first-letter
The first letter
Of an LI element
Whose immediate sibling is an LI element
Whose descendant has an id of "content"

34. form [id="address1"]
An element with the "id" attribute of "address1"
Whose descendant is a FORM element

35. blog > p:last-child
The last child
That is a P element
Whose parent is a BLOG element

36. nav ul * a.active:link
An unvisited link
With the class "active"
That is part of an A element
That has descendants besides UL or NAV
Whose descendent is a UL element
Whose descendent is a NAV element

37. :not([id="#intro"]) h3.small
For a given element h3
Containing the class "small"
Descended from an element that does not have an id attribute equal to "#intro"

38. input#school.professor.student:checked
A checked checkbox
With the class "student"
With the class "professor"
With the id "school"
That is part of a Input element

39. a.special[title]:link
An unvisited link
With the attribute "title"
Containing the class "special"
That is part of an A element

40. a .special[title]:link
An unvisited link
With the attribute "title"
Containing the class "special"
Descended from an a element
Note: According to W3C, an A tag cannot be nested within another A tag, so there is no valid HTML that applies to this.
